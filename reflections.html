What is this Proof of Concept?

This proof of concept is for a wallpaper web app, designed using HTML, CSS, and JavaScript. It fetches wallpaper content and metadata from a JSON file, displaying the images in a visually appealing layout. The project also demonstrates the app’s Progressive Web App (PWA) functionality, allowing users to experience a seamless and mobile-friendly interface.

Why Build This?

The idea for this project was inspired by MKBHD’s wallpaper app. The goal was to explore whether a simpler, more streamlined version could be built and still deliver a satisfying user experience. It serves as an experiment to test the feasibility of building an intuitive and minimalistic wallpaper app that can potentially grow with future features.

Please note that this app is designed specifically with a mobile user interface in mind and is not intended for use on desktop or larger screens.

Project Walkthrough

When you load the app, it takes a few seconds to fetch and display wallpapers from the JSON file. The main page lists all available wallpapers, and by clicking on any wallpaper, users can either download it or close the view to return to the main menu. The download button directly saves the wallpaper without opening the full image. The app includes three core sections:

  1. Main Page – Displays a list of all available wallpapers.
  2. Tags Page – Features a tag cloud where users can browse wallpapers by category or tags.
  3. About Page – Provides background information on the app’s development and purpose.

Initially, there was a plan to add a page for saving favorite tags and categories, but after considering the design, the tag cloud feature was deemed sufficient for helping users easily find wallpapers by their preferred tags.
